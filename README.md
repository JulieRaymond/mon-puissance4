# Puissance 4

Bienvenue dans le projet Puissance 4 ! Ce projet consiste en une implémentation simple du jeu classique Puissance 4 en utilisant HTML, CSS, et JavaScript.
Je l'ai créé seule en parallèle de ma formation développeur bac +2 à la Wild Code School.

## Introduction

Le Puissance 4 est un jeu de stratégie classique dans lequel deux joueurs s'affrontent pour aligner quatre de leurs jetons de la même couleur horizontalement, verticalement ou en diagonale sur une grille de 7 colonnes et 6 lignes.

## Fonctionnalités

- Interface utilisateur simple et interactive.
- Alternance entre les joueurs à chaque coup.
- Détection automatique de la victoire lorsque quatre jetons de la même couleur sont alignés.
- Animation lors du placement des jetons.

## Instructions

1. Ouvrez le fichier `index.html` dans votre navigateur.
2. Cliquez sur une colonne pour placer un jeton.
3. Le jeu alterne automatiquement entre les joueurs.
4. Une alerte s'affiche lorsqu'un joueur gagne.
5. Actualisez la page pour recommencer une partie.

## Structure du Projet

- `index.html`: Fichier principal contenant la structure HTML du jeu.
- `app.js`: Fichier JavaScript gérant la logique du jeu.
- `style.css`: Fichier CSS pour la mise en page et le style.
